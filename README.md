
#  概览

  > 为了给广大用户提供更优质的网络服务和更便捷的组网能力，基于组网产品的发展规划，UDPN产品处在下线周期中，最终将合并于云联网UGN产品。基于以上优化，我们将于2026年01月15日0时起 对 UDPN 的新增场景界面进行调整，具体调整内容如下：
  > 1. 对于未使用过UDPN和云互通产品的新增用户：将关闭UDPN新建通道入口，同时不再支持新建VPC互通资源。新增用户请统一使用功能更强大、体验更优的云联网UGN产品。
  > 2. 对于存量用户：您仍可继续选择使用UDPN产品，但我们会在界面提供引导提示，推荐您尝试和切换至云联网UGN产品，以享受我们最新的网络服务，附：[UGN产品介绍](https://docs.ucloud.cn/ugn/README)。
